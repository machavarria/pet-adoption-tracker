### Description
User can Login/Sign up to make an account for the Dog Adoption Tracker and from there can enter the breed and age for the dog you are looking to adopt.

![](public/img/dog-adoption-screenshot.png)

### Tech Used:

- HTML
- CSS
- Javascript
- MongoDB
- Node.js

### Lessons Learned:
- How to create and manage a database collection with MongoDB
- Applying different parameters for the api
- How to handle form inputs and pass data from the frontend to the backend

## Installation

1. Clone repo
2. run `npm install`

## Usage

1. run `node server.js`
2. Navigate to `localhost:3210`
